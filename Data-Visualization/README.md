# Data Visualization

We do a poor job of spotting trends and relationships among values when data is presented in tables.

As we generate more and more data, Data Visualization becomes even more important.

https://www.tableau.com/learn/articles/data-visualization



## Examples in this project

[data_viz_google_chart.html](./data_viz_google_chart.html) uses the Google Charts API to chart Missouri student count data.

[data_viz_map_of_filtered_data.html](./data_viz_map_of_filtered_data.html) uses the Google Maps API to display information about traffic crashes in Chicago.
