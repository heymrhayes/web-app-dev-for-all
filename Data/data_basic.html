Data - Basic AJAX call

This page makes an AJAX call to the City of Chicago Data Portal to retrieve information related to car crashes. The Portal includes an API that retrieves the data you request and returns it as a JavaScript object that we can work with in our programs.

Script on this page makes the request and logs the response to the console. To see the data, open your browser's developer tools to the JavaScript console; you can also see the request and response on the Network tab. By default, the API limits the response to 1000 records, but with more knowledge of the API, we can override that.

The home page for the dataset is at https://data.cityofchicago.org/Transportation/Traffic-Crashes-Crashes/85ca-t3if/
