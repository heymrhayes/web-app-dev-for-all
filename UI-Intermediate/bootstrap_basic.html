Basic Bootstrap Template

This page includes some files and tags needed to make a Reponsive Web Design page using Bootstrap.

  • Bootstrap CSS
  • Bootstrap JavaScript
  • jQuery library
  • Popper.js

The gray component is a Jumbotron. https://getbootstrap.com/docs/4.3/components/jumbotron/, and inside the Jumbotron is a List Group component. https://getbootstrap.com/docs/4.0/components/list-group/

Bootstrap components include HTML tags which use class attributes to connect CSS style and layout properties with the HTML elements.

It can look very complex, but if you copy and paste the component samples - and only make modifications you understand - you can make functional, attractive and responsive interfaces easily.
